# Файловые системы

Локальные ФС (жёсткие диски, SSD, USB и т.д.) для nix-систем:      
*  Ext (Extended File Systems): семейство стандартных ФС Linux: ext2, ext3 (журналируемая ext2) и широкоиспользуемая ext4 (высокая производительность, повышенная надёжность, поддержка больших файлов);  
*  XFS: выскопроизводительная, масштабируемая ФС с ведениме журнала (разработана SGI, часто используется в крупных компаниях);   
*  Btrfs (B-tree File System): современная ФС, в которой много внимания уделено отказоустойчивости (создание снимков, контроль целостности данных и т.д.);   
*  F2FS (Flash-Friendly File System): ФС, разработанная Samsung, для использования на устройствах флеш-памяти NAND (смартфоны, SSD).   

Локальные ФС DOS/Windows:  
* vfat (FAT12, FAT16, FAT32): длинные названия файлов, совместимость с DOS/Windows;
* ntfs (NTFS3 driver): чтение и запись в Microsoft's primary file system.

Сетевые файловые системы - доступ к файлам на удалённых компьютерах по сети    
* NFS (Network File System): стандартный протокол доступа к файлам в Unix/Linux;
* CIFS (Common Internet File System) / SMB: поддержка протокола SMB доступа к файлам в Windows;
* Ceph / OrangeFS: распределённая ФС для кластеров.
